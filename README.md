# Cipher Decoder App

A modern Python desktop application for encrypting, decrypting, and analyzing Caesar cipher messages using a graphical interface built with CustomTkinter.

## Features
- Encrypt text messages
- Decrypt encrypted messages
- Adjustable shift values (0–25)
- Brute-force cryptanalysis mode
- Copy output to clipboard
- Clear input/output instantly
- Modern dark-themed GUI

## Technologies Used
- Python
- CustomTkinter
- Cryptography basics

## Skills Demonstrated
- GUI application development
- Object-oriented programming (OOP)
- Event handling
- String manipulation
- Caesar cipher implementation
- Brute-force algorithm logic

## How to Run
1. Install dependencies:

pip install customtkinter

2. Run:

python filename.py
